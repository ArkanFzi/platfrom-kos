# Project Kos-Kosan

This project consists of a Next.js frontend and a Golang backend.

## Structure

- `fe/`: Frontend application (Next.js)
- `be/`: Backend application (Golang with Gin & GORM)

## Getting Started

### Backend

1. Navigate to the backend directory: `cd be`
2. Run the server: `go run cmd/api/main.go`
3. The server runs on `http://localhost:8080`

### Frontend

1. Navigate to the frontend directory: `cd fe`
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Access the app at `http://localhost:3000`
